# dotfiles
My personal dotfiles

## Install

Run

	git clone git@github.com:joostd/dotfiles.git ~/.dotfiles
	cd ~/.dotfiles

Symlink to anything needed, e.g

	ln -s .dotfiles/vimrc .vimrc
